# Called to Code

> Building tools and technology for the Adventist community.

## Who We Are

**Called to Code** is a GitHub organization dedicated to creating open-source software that serves the Seventh-day Adventist Church and its ministries. We build apps, APIs, and tools that make Adventist resources—from audiobooks to study materials—more accessible to everyone.
